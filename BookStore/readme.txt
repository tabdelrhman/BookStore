1- Clone the repository into your workspace. When appears in GIT prespective in IDE, Right click on it and select "Import Maven projcet".

2- After the project appears in Projects Explorer, Right click on the project and select "Run As" then "Maven Build" then write in Goal: "clean install -e -X".

3- After the build is success, Right click again and select "Maven" then "update project"

4- Check that the required JARs is downloaded in "Maven Dependancies" folder under the imported project.

5- Open the BookStore.java class and click Run.

6- Program will run. Test all the functionalities.

HINT: I used an Excel file to store and retrieve the data. "Book_Store.xlsx" which is stored in the project directory.
